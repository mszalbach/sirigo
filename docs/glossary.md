# Glossary

| Term        | Definition                                                                                                                          |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| SIRI        | Service interface for real-time information relating to public transport operations                                                 |
| VDV         | Verband Deutscher Verkehrsunternehmen                                                                                               |
| SIRI client | A SIRI client subscribes to data at a SIRI server. The SIRI server will inform the client when data is ready and should be fetched. |
| SIRI server | A SIRI server holds the public transport data like vehicle postions or timetables                                                   |
